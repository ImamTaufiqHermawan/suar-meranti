# Contoh Template Aspirasi — Copy Paste ke Rich Text Editor

Gunakan untuk testing form. **Pilih kategori** sesuai label, lalu **paste HTML** ke kolom "Isi Pesan".

> Cara paste: klik area editor → `Ctrl+V` (Windows) / `Cmd+V` (Mac).  
> TipTap akan mempertahankan format (H1, bold, list, tautan, dll).

---

## 1. SARAN — Penerangan Jalan Cluster

**Kategori:** `Saran`  
**Anonim:** bebas (contoh pakai identitas)

**Nama contoh:** Bpk. Rudi Santoso  
**Alamat contoh:** Blok RE-12, Bukit Meranti

### HTML (copy dari baris `<h1>` sampai penutup `</p>` terakhir)

```html
<h1>Saran Perbaikan Penerangan Jalan</h1>
<h2>Kondisi Saat Ini di Bukit Meranti</h2>
<h3>Area yang Paling Gelap</h3>
<p>Sebagai warga yang <strong>sering pulang malam</strong>, saya ingin menyampaikan <em>saran perbaikan</em> terkait <u>penerangan jalan</u> di beberapa titik cluster.</p>
<p>Beberapa lokasi yang perlu perhatian:</p>
<ul>
  <li>Gang masuk Blok RE-08 sampai RE-12 — lampu redup atau mati</li>
  <li>Tikungan menuju pos security — kurang terang saat hujan</li>
  <li>Jalan menuju area parkir warga — beberapa tiang tidak menyala</li>
</ul>
<p>Usulan langkah yang bisa dikoordinasikan:</p>
<ol>
  <li>Audit titik lampu per blok bersama pengurus cluster</li>
  <li>Laporkan ke pengelola perumahan untuk perbaikan tiang utama</li>
  <li>Pertimbangkan lampu tenaga surya di titik yang sulit dijangkau listrik</li>
  <li>Follow-up hasil perbaikan di grup warga maksimal 2 minggu</li>
</ol>
<p>Referensi lokasi cluster: <a href="https://maps.app.goo.gl/snKgxQBd51bDVZ3u6" target="_blank" rel="noopener noreferrer">Google Maps Bukit Meranti</a></p>
```

## 2. ASPIRASI — Ketua Cluster Baru & Program Warga

**Kategori:** `Aspirasi`  
**Anonim:** bebas

**Nama contoh:** Ibu Siti Rahmawati  
**Alamat contoh:** Blok RE-05, Bukit Meranti

### HTML

```html
<h1>Selamat & Aspirasi untuk Ketua Cluster Baru</h1>
<h2>Momentum Pelantikan Sabtu Kemarin</h2>
<h3>Harapan Program 6 Bulan ke Depan</h3>
<p>Alhamdulillah, <strong>pelantikan ketua cluster Bukit Meranti</strong> pada <em>Sabtu kemarin</em> berjalan <u>khidmat dan tertib</u>. Semoga amanah baru membawa kemajuan bersama.</p>
<p>Aspirasi utama warga:</p>
<ul>
  <li>Komunikasi rutin ketua dengan perwakilan tiap blok RE</li>
  <li>Transparansi iuran dan laporan penggunaan dana cluster</li>
  <li>Program gotong royong lingkungan minimal sebulan sekali</li>
</ul>
<p>Prioritas koordinasi yang diharapkan:</p>
<ol>
  <li>Rapat kenalan pengurus + perwakilan warga minggu ini</li>
  <li>Penetapan kanal aspirasi resmi (SuarMeranti + grup WA)</li>
  <li>Penyusunan agenda jangka pendek: keamanan, air, dan penerangan</li>
  <li>Follow-up pembangunan <strong>musala/masjid</strong> di Meranti — update progress, jadwal, dan kebutuhan sukarela warga</li>
</ol>
<p>Info cluster: <a href="https://www.citraindah.com/cluster/bukit-meranti" target="_blank" rel="noopener noreferrer">Bukit Meranti di CitraIndah.com</a></p>
```

## 3. KELUHAN — Air PDAM & Tembok Perbatasan

**Kategori:** `Keluhan`  
**Anonim:** disarankan anonim jika sensitif

### HTML

```html
<h1>Keluhan Warga: Air PDAM & Tembok Perbatasan</h1>
<h2>Masalah Air Bersih</h2>
<h3>Dampak ke Aktivitas Sehari-hari</h3>
<p>Beberapa hari terakhir <strong>pasokan air PDAM</strong> di area kami <em>tidak stabil</em>: tekanan <u>lemah</u>, kadang mati total di pagi dan sore hari.</p>
<p>Gejala yang dialami warga:</p>
<ul>
  <li>Air keruh saat pertama kali mengalir setelah pemadaman</li>
  <li>Tekanan tidak cukup untuk tangki atas lantai 2</li>
  <li>Jadwal pengisian toren menjadi tidak terprediksi</li>
</ul>
<p>Keluhan kedua — <strong>tembok perbatasan</strong> yang terhubung dengan pedesaan di belakang cluster:</p>
<ol>
  <li>Ada bagian tembok yang belum ditutup/dipasang dengan benar</li>
  <li>Celah memungkinkan akses liar dan meningkatkan kekhawatiran keamanan</li>
  <li>Perlu verifikasi lapangan bersama security dan pengurus cluster</li>
  <li>Mohon estimasi waktu perbaikan dan penanggung jawabnya</li>
</ol>
<p>Lokasi cluster untuk penandaan titik masalah: <a href="https://maps.app.goo.gl/snKgxQBd51bDVZ3u6" target="_blank" rel="noopener noreferrer">Peta Bukit Meranti</a></p>
```

## 4. PUJIAN — Security & Ronda Malam

**Kategori:** `Pujian`  
**Anonim:** bebas

**Nama contoh:** Bpk. Agus Wijaya  
**Alamat contoh:** Blok RE-18, Bukit Meranti

### HTML

```html
<h1>Apresiasi untuk Tim Security Cluster</h1>
<h2>Ronda & Patroli Malam Hari</h2>
<h3>Laporan ke Pengurus yang Konsisten</h3>
<p>Saya ingin menyampaikan <strong>pujian</strong> kepada tim <em>security</em> yang <u>berjaga keliling lingkungan</u> terutama pada malam hari. Kerja mereka terasa dan memberi rasa aman.</p>
<p>Hal yang patut diapresiasi:</p>
<ul>
  <li>Patroli berkala ke titik rawan dan gang sempit</li>
  <li>Sikap ramah namun tegas saat mengarahkan tamu/penjual keliling</li>
  <li>Rela mengecek laporan warga meski sudah larut malam</li>
</ul>
<p>Alur kerja yang sudah bagus menurut saya:</p>
<ol>
  <li>Security menerima laporan dari warga via pos atau WA</li>
  <li>Tim patroli mengecek lokasi dan update kondisi</li>
  <li>Hasil dilaporkan ke ketua/anggota cluster untuk tindak lanjut</li>
  <li>Esok harinya ada ringkasan singkat ke grup warga (jika perlu)</li>
</ol>
<p>Semoga <strong>standar keamanan</strong> ini <em>dipertahankan</em> dan <u>ditingkatkan</u> bersama pengurus baru. Terima kasih!</p>
```
